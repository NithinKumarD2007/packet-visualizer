# Packet Visualizer

AI-powered packet analyzer dashboard built using:

- Python
- Scapy
- SQL Server
- Tkinter
- Google Gemini AI

## Features

- Live Packet Capture
- Protocol Statistics
- Pie Chart Visualization
- Threat Level Detection
- AI Network Assistant
- SQL Server Storage
